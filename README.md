# 3DM Auth UI

فرم لاگین و ثبت‌نام مدرن با طراحی حرفه‌ای، انیمیشن‌های جذاب و پیام‌های Toast  
ساخته شده با **HTML + TailwindCSS + javascript + FontAwesome**

---

## ✨ ویژگی‌ها

- طراحی کاملاً ریسپانسیو و مدرن
- استفاده از پالت رنگی اختصاصی و گرادیانت
- انیمیشن جابجایی بین صفحات (Flip 3D)
- پیام‌های خطا و موفقیت به صورت Toast با افکت Fade-in
- دکمه‌های ورود با شبکه‌های اجتماعی
- UX حرفه‌ای و کدنویسی تمیز و پوشه‌بندی شده
- بدون نیاز به فریم‌ورک (Pure HTML/CSS/JS)

## ⚡️ امکانات ویژه

- **انیمیشن جابجایی:** بین صفحات لاگین و ثبت‌نام بدون ریلود و با افکت Flip
- **Toast:** پیام خطا (مثلاً ایمیل/پسورد اشتباه) و موفقیت (ورود/ثبت‌نام موفق) با افکت و رنگ‌بندی جذاب
- **پالت رنگی:** رنگ‌ها و گرادیانت‌ها مطابق UI مدرن و تصویر مرجع
- **کدنویسی تمیز:** استفاده از متغیرهای CSS و کلاس‌های سفارشی برای توسعه‌پذیری بالا
